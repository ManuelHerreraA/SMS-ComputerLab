# SMS Computer Lab: Signal Processing
This repository contains the implementation of the Signal Processing Computer Laboratory (Universit√© Paris-Saclay).

## Content
- **4.1. Warmup:** Basic Python and numerical calculus.
- **4.2. Basic Fourier Properties:** Analysis of Gate and Gaussian functions.
- **4.3. Windowing:** Reconstruction of signals and effect of windowing.
- **4.4. Convolution, Filtering:** Implementation of convolution kernels. 
- **4.5. Audio Signal Filtering:** Implementation of RC and RLC filters to noise extraction.
- **5.1. Audio Signal Compression:** Compression algorithm for sound signals (no yet finished)
- **5.2. Image Analysis:** 2D FFT for different images, and attempt for denoising.
- **5.3. WFT:** Spectrograms, piano note detection, and voice pitch shifting.

## Tools
- Python 3
- NumPy, Matplotlib
